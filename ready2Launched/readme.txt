
Socca is ready to launched if:

1) The twitter4j.properties is located next to the Socca.jar

2) All *.jar (Socnetsdkv2 and etc) are inside Lib folder

3) Put socca (system) directory
under AppData\Roaming containing:
- audios : horray.wav, pop.wav, ticktock.wav
- database : database.db

3) See the preview on these images:
- properSystemFolderFile01.PNG
- properSystemFolderFile02.PNG
- properSystemFolderFile03.PNG

4) Make a shortcut on Desktop for Socca.jar
and all ready to go!
